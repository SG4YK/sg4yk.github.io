## Level4
![](/assets/img/post/2019/10/1024_4.png)
非常简单的程序题，不用考虑超时问题，直接暴力枚举即可  
Java示例代码
~~~java
import java.util.Scanner;

public class Main {

    public static void main(String[] args) {
        int n = 1;
        int i = 1009;
        while (n < 1024) {
            if(match(++i)) {
                n++;
            }
        }
        System.out.println(i);
    }

    public static boolean match(int x) {
        for (int i = 2; i < 1000; i++) {
            if (x % i == 0) {
                return false;
            }
        }
        return true;
    }
}
~~~
**flag**  <code>9649</code>
